    ## 💡 Intuition

Each element in the input array can either be included or excluded from a subset, leading to 2^n total subsets. We can represent each subset using a unique bitmask where the j-th bit indicates the inclusion of the j-th element.

## 🧩 Approach

Iterate from 0 to 2^n - 1, where n is the size of the input array. Each integer 'i' in this range represents a unique subset. For each 'i', iterate through the elements of the input array 'nums' using index 'j'. If the j-th bit of 'i' is set (i.e., (i & (1 << j)) is true), include nums[j] in the current subset. Collect all such elements into a temporary vector and add it to the final result list.

## ⏱️ Complexity

- **Time:** O(n * 2^n), because there are 2^n subsets and for each subset, we iterate up to n times to check bits and build the subset.
- **Space:** O(n * 2^n), to store all 2^n subsets, each potentially containing up to n elements.

## ⚠️ Edge Cases

The solution naturally handles the case of an empty set (when i=0, no bits are set) and single-element inputs. The problem constraints guarantee unique elements and a non-empty input array, simplifying considerations.
## Problem Statement

Given an integer array nums of unique elements, return all possible subsets (the power set).

The solution set must not contain duplicate subsets. Return the solution in any order.

 
Example 1:

Input: nums = [1,2,3]
Output: [[],[1],[2],[1,2],[3],[1,3],[2,3],[1,2,3]]


Example 2:

Input: nums = [0]
Output: [[],[0]]


 
Constraints:


	1 <= nums.length <= 10
	-10 <= nums[i] <= 10
	All the numbers of nums are unique.
